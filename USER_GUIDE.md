# Student Performance Tracker - User Guide

## Introduction
Student Performance Tracker is a Flask-based web application used to manage and analyze student academic performance.

## How to Use the Application

### 1. Add Student
- Open the application homepage.
- Navigate to Add Student.
- Enter:
  - Student name
  - Roll number
  - Subject marks
- Click Submit.
- The system calculates average marks and grade automatically.

### 2. View Students
- Open Student List.
- View all student records with marks, average, and grade.

### 3. Edit Student Details
- Click Edit.
- Update student information.
- Save changes.

### 4. Delete Student
- Click Delete.
- Confirm removal of student record.

### 5. Search Student
- Search using:
  - Roll number
  - Student name

### 6. Dashboard
The dashboard displays:
- Total students
- Class average
- Top performer
- Lowest performer
- Grade distribution
- Subject averages

## Technologies Used
- Python
- Flask
- SQLite
- HTML/CSS
- Jinja2 Templates
